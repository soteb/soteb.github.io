一划深思 Chrome 扩展程序隐私权政策

单一用途声明

一划深思是一个专注于文本翻译和深度理解的 Chrome 扩展程序。其单一且明确的用途是：帮助用户快速翻译和深入理解网页中选中的文本内容。用户自行配置API KEY通过集成先进的 AI 技术，为用户提供准确的翻译和深度分析，让用户能更好地理解和学习网页内容。

隐私权政策

1. 信息收集

本扩展程序不会收集任何个人身份信息。我们只会处理以下必要信息：

  • 用户选中的文本内容（仅用于实时翻译和分析）
  • 用户的翻译历史记录（仅存储在用户本地设备中）
  • API 密钥（用户自行提供，仅用于访问 AI 服务）

2. 信息使用

  • 所有的文本处理都是即时的，不会在服务器端保存
  • 翻译历史记录仅保存在用户的本地浏览器存储中
  • 用户可以随时清除历史记录

3. 数据存储

  • 所有数据仅存储在用户本地设备中
  • 不会将任何数据上传到云端或第三方服务器
  • 用户可以完全控制数据的保存和删除

4. 第三方服务

本扩展程序可能会使用以下第三方服务：

  • Kimi、DeepSeek、Qwen API（仅当用户提供自己的 API 密钥时）

5. 数据安全

  • 所有的 API 密钥都经过加密存储在用户本地
  • 不会收集或传输任何用户的个人信息
  • 所有的网络请求都使用安全的 HTTPS 协议

6. 用户权利

用户拥有以下权利：

  • 随时查看和删除本地存储的翻译历史
  • 完全控制 API 密钥的添加和删除
  • 随时停用或卸载扩展程序

7. 政策更新

我们可能会不时更新本隐私权政策。重大变更时，我们会通过扩展程序界面通知用户。

8. 联系我们

如果您对本隐私权政策有任何疑问，请通过 Chrome 网上应用店的开发者联系方式与我们联系。
